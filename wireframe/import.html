How to Export Data

IG Instagram
  1. Go to Professional Dashboard
  2. Click "Insights"
  3. Select date range
  4. Click "Export" → CSV
FB Facebook
  1. Open Meta Business Suite
  2. Go to Insights
  3. Click "Export Data"
  4. Choose CSV format
TT TikTok
  1. Open TikTok Studio
  2. Go to Analytics
  3. Click "Download data"
  4. Select date range
LI LinkedIn
  1. Go to Page Admin view
  2. Click "Analytics"
  3. Select metric type
  4. Click "Export"
